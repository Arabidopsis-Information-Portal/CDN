JavaScript, Cascading Style Sheet Content Delivery Network

Maintain resources used in InterMine apps

Add new resource

  1. Add your library as per conventions in this repo
  2. Add a package.json and validate through JSONLint
  3. Commit
  4. git pull latest changes on the server

Where?

The repo is cloned on met0 powered by nginx on port 1000 and should be accessible through http://cdn.intermine.org
